## Part 3. From Occupancy Data to Space Use Indicators

Part 3 mainly focuses on how to turn occupancy data into space use indicators. In other words, we want to change the raw data into information that is more useful for our analysis.

At the moment, the most direct information from our occupancy data is how many people are in a room at a certain time. However, we cannot only look at this number, because it does not directly tell us whether the room is being fully used or not.

The literature also gives us some useful ideas. For example, we can create some simple indicators, such as average occupancy, peak occupancy, empty time, and utilisation rate.

Also, some studies remind us that sensor data may not always be fully reliable. So, we still need to do basic data quality checks before analysis. For example, we need to check missing values, abnormal values etc. This is also related to our previous discussion about the value column, because we have already done some basic checks and discussed how to understand negative values and abnormal values.

So, the main point of Part 3 is that we should not only describe the raw occupancy numbers. We need to turn the data into more useful information. This will help us compare the use of different rooms, different time periods, and different buildings later.

---

## Part 4. Practical Use of Occupancy Analysis

Part 4 mainly talks about the practical use of occupancy analysis.

After we understand the occupancy changes across different rooms and different time periods, and after we create some useful space use indicators, these results can help the university make better space management decisions.

For example, the university can know which rooms often have low utilisation, which time periods are peak times, which spaces may need to be reallocated, and whether the timetable arrangement can be improved.

Some papers also connect occupancy information with energy saving. These papers give us an idea that occupancy analysis can provide some support for future energy-saving directions.

For example, if we find that some rooms are empty for many time periods, or some buildings have low usage during certain times, the university may be able to arrange air conditioning, lighting, and ventilation more efficiently in the future.

So, our understanding is that occupancy analysis is not only for describing data or making charts. It can also provide a foundation for future campus space management and possible energy-saving plans.
